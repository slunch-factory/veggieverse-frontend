## 우선 보강 항목 (무료·코드로 바로 가능)

1. **구독 정기결제 조항** (약관 + 환불정책) — 갱신 고지/중도 해지/잔여분 환불
2. **개인정보 파기 절차·방법** 항목 추가 (처리방침)
3. **만 14세 미만 아동 미수집** 명시 (처리방침)
4. **미성년자 결제** 조항 (약관)
5. 호스팅사업자(Vercel) 표시 문구 확인

→ 위 1~5는 표준 양식에 다 있는 문구라, 표준약관/PIPC 도구에서 가져와 채우면 됨(추가 비용 없음).

## 무료 상담에서 콕 집어 물어볼 것 (비표준 부분)

- 구독(정기결제)의 **사전 고지·해지 절차**가 충분한가? (정기결제 규제 강화 추세)
