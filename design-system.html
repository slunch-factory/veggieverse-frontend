02 · FOUNDATION

Typography

기본 서체는 Noto Sans KR. 본 시스템은 가독성과 균형을 위해 대부분의 텍스트를 Regular(400) 웨이트로 구성하며, 강조가 필요한 타이틀에만 Medium(500) / SemiBold(600)을 사용합니다.

display AI 기반 비건 레시피 40 / 1.3 / -3%
h1 오늘의 맞춤 식단 32 / 1.3 / -2%
h2 구독 플래너 구성 24 / 1.4
h3 주차별 메뉴 선택 18 / 1.5
body 식단 구성 후 배송 주기를 선택하면 자동으로 견적이 산정됩니다. 14 / 1.7
small 최소 주문 수량 및 변경 정책을 확인해 주세요. 13 / 1.6
caption * 일부 품목은 계절에 따라 구성이 달라질 수 있습니다. 11 / 1.5

Button

사용자의 액션을 유도하는 기본 컨트롤. 변형(Variant) · 크기(Size) · 상태(State) 세 축으로 토큰이 구성되어 있습니다.

Anatomy · 구성 요소
⊕ 버튼 라벨 ＋
① Container구성 요소를 담는 컨테이너
② Label버튼 텍스트
③ Leading Element앞쪽에 위치하는 아이콘
④ Trailing Element뒤쪽에 위치하는 아이콘

Primary · Secondary · Outline · Danger · Ghost
Sizes · 크기

// components/ui/Button.tsx

const button = cva(
  "inline-flex items-center justify-center gap-2 font-medium transition-colors focus-visible:outline-2 focus-visible:outline-primary focus-visible:outline-offset-2",
  {
    variants: {
      variant: {
        primary: "bg-primary text-white hover:bg-primary-dark active:bg-primary-darker disabled:bg-line disabled:text-muted",
        secondary: "bg-primary-50 text-primary-dark hover:bg-primary-100",
        outline: "border border-line text-charcoal hover:bg-eggshell",
        danger: "bg-tomato text-white hover:bg-tomato-dark",
        ghost: "text-charcoal hover:bg-eggshell",
      },
      size: {
        lg: "h-12 px-5 text-[15px] rounded-lg",
        md: "h-10 px-4 text-sm rounded-md",
        sm: "h-8 px-3 text-[13px] rounded-md",
      },
    },
    defaultVariants: { variant: "primary", size: "md" },
  }
);

13 · USAGE

Do & Don't

참고 문헌의 "UI는 형태가 아닌 목적"이라는 원칙을 기준으로, Primary 버튼은 한 화면에 하나만 사용하고, 파괴적 액션(삭제·취소)은 반드시 Danger 변형으로 표기합니다.

✓ Do

한 화면 내 Primary 1개, 보조 액션은 Outline/Ghost로.

× Don't

Primary 3개를 동시에 배치하지 않습니다 — 시선이 분산됩니다.
